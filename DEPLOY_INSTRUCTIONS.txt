PROPERTYSHIELD ROOFING - DEPLOYMENT INSTRUCTIONS
=================================================

REQUIREMENTS:
  - Node.js 20+
  - PostgreSQL database
  - npm

STEPS TO DEPLOY ON A NEW HOST:
--------------------------------

1. UPLOAD YOUR CODE
   - Download the ZIP of this project from Replit (three dots menu -> Download as ZIP)
   - Upload/extract it on your new server

2. INSTALL DEPENDENCIES
   Run: npm install

3. SET UP YOUR DATABASE
   - Create a PostgreSQL database on your new host
   - Run the database.sql file included here to create the tables:
     psql YOUR_DATABASE_URL < database.sql

4. CREATE YOUR .env FILE
   - Copy .env.example to .env
   - Fill in your DATABASE_URL from your new PostgreSQL host
   - Set a random SESSION_SECRET

5. BUILD THE FRONTEND
   Run: npm run build

6. START THE SERVER
   Run: npm start
   (The server runs on port 5000 by default)

HOSTING OPTIONS THAT WORK WELL:
  - Render.com (free tier available, easy setup)
  - Railway.app (includes free PostgreSQL)
  - Any Linux VPS (IONOS VPS, DigitalOcean, etc.)
